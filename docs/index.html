▼このサイトについて

「文芸風前会」所属メンバーの一人、出汁殻 ニボシが主に活動報告の場として開設したサイト。
「意識低めの意欲高め」をそれとなく標榜しつつ、週に1回は更新していきたい。
文章や作品などは上の方にあるカテゴリタブからお読みいただけます。

▼連絡先

Twitter

ゲームの制作記録や雑記など、テキストを主体としたコンテンツ。
以下のブロックから各記事一覧へ飛べるほか、
小説などの創作は上タブの「リンク」から文芸風前会、
カクヨムなどの各種サイトでもお読みいただけます。
